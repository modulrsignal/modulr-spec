# Interoperability

Modulr supply descriptors are designed to be consumed by systems operating within existing advertising infrastructure.

This section documents where Modulr fits relative to common ecosystem components. These are not integration guides. They are positioning documents that answer three questions for each component:

1. What responsibility does this component own?
2. Where does Modulr fit relative to it?
3. What data crosses the boundary?
