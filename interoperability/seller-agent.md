# Seller Agent

## What a seller agent owns

A seller agent represents publisher inventory in programmatic environments. It manages supply availability, floor logic, deal targeting, and auction participation on behalf of publishers.

## Where Modulr fits

Modulr descriptors provide machine-readable supply intelligence that seller agents can consume to enrich their inventory representation.

Modulr does not set floors. Modulr does not manage deals. Modulr does not participate in auction logic.

The descriptor tells the seller agent what the supply means. The seller agent decides how to use that meaning in its own execution context.

## What crosses the boundary

Supply descriptor fields — particularly topics, intent signals, suitability, and confidence — are available as structured signals that seller agents can attach to inventory objects or use in targeting and filtering logic.
